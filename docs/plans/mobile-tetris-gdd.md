## 🎮 게임 개요

**장르:** 퍼즐 (테트리스 변형)
**플랫폼:** PixiJS (모바일 웹 브라우저 - 세로 모드)
**타겟 플레이어:** 캐주얼 게이머, 출퇴근 시간에 즐기는 직장인
**플레이 시간:** 세션당 3-5분

**한 줄 요약:**
한 손으로 즐기는 세로 화면 테트리스 - 직관적인 제스처 컨트롤

## 🎯 핵심 경험 (Core Experience)

플레이어가 느껴야 할 **주요 감정/경험**:
- 라인 클리어의 만족감과 쾌감
- 짧은 집중 시간 동안의 몰입감
- "한 판만 더" 중독성

**모바일 특화 경험:**
- 한 손으로 편안하게 플레이
- 대기 시간 활용 (지하철, 버스)

## 🕹️ 핵심 메커니즘 (Core Mechanics)

### 1. **제스처 기반 컨트롤 (핵심 차별화)**
   - **좌우 드래그:** 블록 좌우 이동
   - **스와이프 다운:** 블록 즉시 낙하 (Hard Drop)
   - **탭(한 번):** 블록 시계방향 회전
   - **더블 탭:** 블록 반시계방향 회전
   - **홀드 존 탭:** 블록 보관 (Hold)

### 2. **라인 클리어 시스템**
   - 기본 테트리스 규칙 유지
   - **콤보 시스템:** 연속 라인 클리어 시 점수 배수 증가
   - **특수 클리어:**
     - 싱글 (1줄): 100점
     - 더블 (2줄): 300점
     - 트리플 (3줄): 500점
     - 테트리스 (4줄): 800점 + "TETRIS!" 효과

### 3. **난이도 증가**
   - 레벨이 오를수록 블록 낙하 속도 증가
   - 10라인 클리어마다 레벨업

### 게임 루프:
```
블록 생성 → 제스처로 이동/회전 → Hard Drop → 라인 클리어 →
점수/콤보 → 다음 블록 → 반복
```

## 📱 기술 스택 (PixiJS 특화)

### 기본 설정
- **PixiJS 버전:** v8.x
- **해상도:** 360x640 (세로 모드, 16:9 비율)
- **타겟 FPS:** 60

### 모바일 최적화
- **터치 입력:** PixiJS InteractionManager + 제스처 라이브러리 (Hammer.js 고려)
- **반응형:** 다양한 화면 크기 대응 (iPhone SE ~ iPad)
- **성능:**
  - 블록: 최대 250개 동시 (게임 오버 시)
  - 파티클: 라인 클리어 당 50개 제한
  - 스프라이트 배칭 사용

### 아키텍처
- **물리엔진:** 없음 (그리드 기반 로직)
- **상태 관리:** 단순 FSM (Menu → Playing → GameOver)
- **에셋 로딩:** PixiJS Assets API

## 🔧 기술 노트 (구현 힌트)

### 제스처 감지 예시:
```javascript
// 드래그로 블록 이동
let dragStartX = null;
container.on('touchstart', (e) => {
  dragStartX = e.data.global.x;
});

container.on('touchmove', (e) => {
  const deltaX = e.data.global.x - dragStartX;
  if (Math.abs(deltaX) > 30) { // 30px 이상 이동 시
    if (deltaX > 0) moveRight();
    else moveLeft();
    dragStartX = e.data.global.x;
  }
});

// 스와이프 다운 감지
let dragStartY = null;
container.on('touchstart', (e) => {
  dragStartY = e.data.global.y;
});

container.on('touchend', (e) => {
  const deltaY = e.data.global.y - dragStartY;
  if (deltaY > 100) { // 빠른 아래 스와이프
    hardDrop();
  }
});

// 탭으로 회전
let lastTapTime = 0;
container.on('tap', (e) => {
  const now = Date.now();
  if (now - lastTapTime < 300) {
    // 더블 탭: 반시계방향
    rotateCounterClockwise();
  } else {
    // 싱글 탭: 시계방향
    rotateClockwise();
  }
  lastTapTime = now;
});
```

### 진동 피드백:
```javascript
function vibrate(duration) {
  if ('vibrate' in navigator) {
    navigator.vibrate(duration);
  }
}

// 사용 예:
onLineClear(() => vibrate(30));
onTetris(() => vibrate(80));
onGameOver(() => vibrate(150));
```
