**목표:** 핵심 메커니즘 검증

- [ ] **검증 기준:** 5분 플레이로 "재미있는지" 판단 가능

**목표:** 완성된 경험의 작은 버전
- [ ] 1개 완성 레벨
- [ ] 기본 UI/메뉴
- [ ] 사운드/비주얼 초안
- [ ] **검증 기준:** "이게 최종 게임의 느낌이다" 전달 가능

### Phase 3: 콘텐츠 제작 (Week 5-8)
**목표:** 모든 레벨/콘텐츠 완성
- [ ] 전체 레벨 구현
- [ ] 모든 적/장애물
- [ ] UI 폴리싱
- [ ] **검증 기준:** 처음부터 끝까지 플레이 가능

### Phase 4: 폴리싱 (Week 9-10)
